## 📌 Sobre o Projeto

O **JobbyHub** é um ecossistema construído em torno de uma API inteligente focada em otimizar o processo de recrutamento para empresas e acelerar candidaturas para usuários.

A proposta do projeto é reduzir o tempo perdido tanto por recrutadores quanto por candidatos utilizando:

* Scraping inteligente de vagas
* Filtragem automatizada
* Integração com IA
* Personalização de currículos
* Automação de candidaturas

A API pode ser integrada em diversos ambientes, como:

* Sites
* Bots
* Sistemas corporativos
* Plataformas de recrutamento

---

## 🎯 Objetivos

### 👨‍💼 Para Empresas

* Reduzir tempo gasto analisando currículos
* Melhorar a filtragem de candidatos
* Automatizar processos repetitivos
* Encontrar candidatos mais compatíveis

### 👨‍🎓 Para Candidatos

* Evitar vagas fantasmas ou encerradas
* Aplicar para múltiplas vagas rapidamente
* Melhorar currículos para ATS
* Receber vagas alinhadas ao perfil
